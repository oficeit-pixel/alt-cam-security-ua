Data Deletion Instructions

Last updated: July 16, 2026

If you want ALT-CAM Security UA to delete personal information you previously provided to us, please send a deletion request by email.

How to request deletion

  1. Email us at altcam.ua@gmail.com.
  2. Use the subject line: Data Deletion Request.
  3. Include the contact details you used when communicating with us, so we can identify the relevant record.

What happens next

We will review the request and delete or anonymize the applicable information unless we need to keep it for legal, accounting, fraud prevention, or security reasons.

Meta app data

Our Meta application is used by ALT-CAM Security UA administrators to publish our own company content. It is not designed to collect private user profile data from customers or followers.
