# Alt-Cam Security UA

Односторінковий адаптивний лендинг українською мовою. Для запуску достатньо завантажити файли `index.html`, `styles.css` і `script.js` в одну папку на хостингу.

## Перед публікацією

Відкрийте початок `script.js` і замініть значення у блоці `CONTACTS`:

- `telegram` — ім’я користувача без `@`;
- `whatsapp` — номер лише цифрами, наприклад `380XXXXXXXXX`;
- `phone` та `phoneLabel` — номер і його підпис на сайті;
- `email`, `facebook`, `instagram`, `tiktok` — реальні контакти проєкту.

Поки Telegram і WhatsApp не вказані, кнопки відкривають стандартне меню поширення месенджера з уже сформованим текстом заявки.

Сайт не потребує backend: після заповнення форми заявка передається у вибраний месенджер. Для роботи шрифту Manrope потрібне інтернет-з’єднання; якщо Google Fonts недоступний, браузер використає системний шрифт без порушення верстки.

## Калькулятор

На сайті працюють три режими калькулятора:

- відеоспостереження — площа, периметр, висота монтажу, роздільна здатність і глибина архіву;
- резервне живлення — квартира, будинок або офіс, вибрані споживачі й потрібний час автономної роботи;
- Ajax та комплексна безпека — площа, поверхи, двері, вікна, пожежні датчики й захист від протікання.

Результати є попередньою оцінкою, а не публічною офертою.

Орієнтовні ціни обладнання та робіт знаходяться у функції `calculateSecuritySystem()` у файлі `script.js`. Їх можна змінити відповідно до актуального прайсу без редагування верстки.

## Перед фінальною публікацією версії 2.0

- Замініть демонстраційні відгуки на перевірені відгуки реальних клієнтів.
- Додайте секцію власних робіт після того, як матимете 10–15 якісних фотографій виконаних об’єктів.
- Вкажіть реальні Telegram, Messenger, Viber, телефон та соціальні мережі у `CONTACTS`.
- Замініть `https://alt-cam.ua/` у `index.html`, `robots.txt` і `sitemap.xml`, якщо сайт буде розміщений на іншому домені.
- Оновіть телефон, email і посилання у JSON-LD блоці `LocalBusiness` в `index.html`.

Стокові фотографії не використовуються.

## CRM та аналітика

На початку `script.js` є блок `INTEGRATIONS`:

- `crmWebhook` — адреса Google Apps Script або CRM webhook для одночасної передачі заявок у Google Sheets, Telegram та Email;
- `ga4Id` — ідентифікатор Google Analytics 4;
- `metaPixelId` — ідентифікатор Meta Pixel;
- `clarityId` — ідентифікатор Microsoft Clarity.

Meta Conversion API потребує серверного або CRM-шлюзу і не може безпечно працювати лише у статичному браузерному коді.

Стартові ціни у каталозі необхідно звірити з актуальним прайсом перед запуском реклами.
